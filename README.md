
# Real vs. Simulated Remote Sensing Data Validation
## This repository is dedicated to the assessment, benchmarking, and validation of simulated remote sensing datasets against real-world Earth observation data. By comparing simulated outputs with real-world collections, this project aims to evaluate the fidelity of synthetic scene modeling architectures (such as DIRSIG) across multiple sensor modalities.

# Repository Structure
The repository is organized by data acquisition platform and sensor type:
## Simulated VS Real ANG Validation Repos
### Focuses on airborne imaging spectroscopy (e.g., AVIRIS-NG style hyperspectral datasets).
### Contains workflows for preprocessing, spectral calibration, and analytical comparisons.

## Simulated VS Real Drone Validation Repos
### Dedicated to high-resolution Unmanned Aerial Vehicle (UAV) datasets.
### Evaluates fine-scale spatial and structural representations.

## Simulated VS Real Lidar data Validation
### Focuses on 3D radiative transfer modeling and structural metrics.
### Validates synthetic point clouds and canopy structure against real-world LiDAR returns.
